🚀 Hexo 简介

Hexo 是一款基于 Node.js 的静态博客框架，支持 Markdown 写作，主题丰富，部署便捷，非常适合技术博客和个人笔记。

📝 常用操作速查

新建文章

新文章会自动生成在 source/_posts/ 目录下。

启动本地预览

访问 http://localhost:4000 查看效果。

生成静态文件

会在 public/ 目录生成静态网页。

部署到远程

需提前配置好 _config.yml 的 deploy 相关参数。
